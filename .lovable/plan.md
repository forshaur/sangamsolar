

# Solar Subsidy Startup Landing Page

Build a complete single-page landing page with eco-friendly design (greens, sun-yellows, slate blues, clean whites) and Inter font.

## Design System Updates
- Update CSS variables for eco-friendly palette: green primary, sun-yellow accent, slate text
- Add Inter font import

## Components to Build

### Navbar
- Sticky header with logo text, nav links (Benefits, Our Process, Contact), and green "Get Started" CTA
- Mobile hamburger menu with sheet/drawer

### Hero Section
- Bold headline about accessible solar + subsidies
- Subheadline about end-to-end service
- "Check Your Subsidy Eligibility" green CTA button
- Decorative sun/solar visual elements (CSS gradients/shapes)

### Benefits Section (Why Go Solar?)
- 4-card grid with Lucide icons:
  - Reduced electricity bills (Zap icon)
  - Government subsidies (BadgeDollarSign icon)
  - Increased property value (Home icon)
  - Carbon footprint reduction (Leaf icon)

### Process Section (How We Help)
- 4-step vertical timeline with numbered steps and icons:
  1. Free Consultation
  2. Subsidy Approval
  3. Professional Installation
  4. Switch On & Save

### Contact Section
- Contact form with fields: Name, Email, Phone, Zip Code, Message
- Form validation and toast feedback on submit
- Placeholder contact info (email, phone, address)

### Footer
- Copyright, privacy policy link placeholder
- Social media icon placeholders (Lucide icons)

## Technical Details
- All sections as components within a single Index page
- Smooth scroll via anchor links
- Fully responsive mobile-first design
- Remove default App.css styles
